# Gamerbolt Clan Name Generator

AI-powered gaming clan name generator with instant generation and trending styles.

## Features

- **AI-Generated Names** - Unique clan names powered by OpenAI GPT-4o-mini
- **Trending Tags** - Quick generation with popular styles (Elite, Funny, Pro, Legendary)
- **Copy All** - Copy all generated names at once
- **Save Favorites** - Save your favorite clan names
- **Smooth UX** - Fast, responsive interface with animations
- **Customizable** - Choose name type, length, and keywords

## Getting Started

### Prerequisites

- Node.js 18+ or pnpm
- OpenAI API key

### Installation

```bash
pnpm install
```

### Environment Variables

Create a `.env.local` file:

```
OPENAI_API_KEY=your_openai_api_key_here
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

### Build

```bash
pnpm build
```

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
